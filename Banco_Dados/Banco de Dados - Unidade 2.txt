Modelo ER(Entidade Relacionamento) 

Entidade 

-Modelo de dados conceitual de alto nível
-Entidade: elemento do mundo real com existência própria (objeto que eu quero modelar)
-Atributo e valor de atributo: propriedades que descrevem a entidade
-Atributo Composto x Simples
-Atributo Monovalorado(único valor para uma entidade particular) x Multivalorado(pode ter um conjunto de valores para uma mesma entidade)
-Atributo Derivado ou Virtual
-Valores NULL
-Atributo chave: identifica cada entidade unicamente
-Domínio de um atributo: valor possível do atributo

Modelo ER(Entidade Relacionamento) 

Relacionamento

-Conjunto de associações entre entidades
-Grau de um Tipo de Relacionamento
-Relacionamento Recursivo: a entidade tá relacionada com a própria entidade, importante falar o papel

->Propriedades do Relacionamento
-Restrição de Cardinalidade: quantidade (1:1,1:N; N:1,N:N)
-Restrição de Participação: total, parcial
