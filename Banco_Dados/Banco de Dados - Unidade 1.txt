-----#-----#-----#-----
Introdução aos Sistemas Gerenciadores de Banco de Dados - SGBD

-Software 
-Abstrair para o profissional de computação o acesso ao dados
-SBD = SGBD+BD

->Características
-Controle de redundâncias
-Compartilhamento de dados
-Restrição de acesso (segurança)
-Múltiplas interfaces
-Forçar restrições de integridade (Ex: padrões de hora, data, moeda etc)
-Backup e recuperação contra falhas
-Controle de transação (conjunto de operações com lógica)
-Tempo de desenvolvimento reduzido (SQL é simples, abstrações de manipulação de arquivos)

-SGBD x Sistema de Arquivos
