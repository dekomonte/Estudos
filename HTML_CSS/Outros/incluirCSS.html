Formas de incluir CSS no HTML. Dá pra colocar as regras de CSS na parte do head, na própria abertura de tag e em um arquivo externo.

Vantagens de Vincular um arquivo CSS externo:

Performance. Quando você vincula um arquivo CSS externo ao projeto, o navegador pode armazenar em cache esse arquivo.

Manutenção. Se você precisar fazer alterações nos estilos, precisa modificar um único arquivo.

Performance. Arquivos HTML mais leves.
